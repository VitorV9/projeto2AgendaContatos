# Agenda de Contatos

Este é um projeto de uma aplicação web de agenda de contatos, desenvolvida como atividade prática de desenvolvimento front-end. O objetivo é permitir que o usuário cadastre nomes e números de telefone, visualize-os em uma tabela organizada e acompanhe a quantidade total de registros.

## 🚀 Tecnologias Utilizadas
- **HTML5**: Estrutura semântica da página.
- **CSS3**: Estilização, layout flexível (Flexbox) e design responsivo.
- **JavaScript**: Lógica de manipulação do DOM, validação de dados e interação.

1. Clone o repositório: `git clone https://github.com/VitorV9/projeto2AgendaContatos.git`
2. Abra a pasta do projeto no seu editor de código.
3. Abra o arquivo `index.html` no seu navegador (ou use a extensão Live Server).
